How to Use

  • Type your main task on the first line (Level 0)
  • All other tasks must be subtasks (Level 1-4)
  • Press Enter to create a new task
  • Press Tab to indent a task (create a subtask)
  • Press Shift + Tab to outdent a task
  • Tasks can be nested up to 4 levels
  • First letter of each task is automatically capitalized
  • Level 0: Main task (only one allowed)
  • Level 1: Subtasks
  • Level 2: Sub-subtasks
  • Level 3: Deep subtasks
  • Level 4: Deepest subtasks

Enhanced Editing

Backspace at indentation: Remove line if empty, otherwise prevent deletion

Delete at line end: Remove empty line below, otherwise prevent newline deletion

Shift+Tab: Smart outdenting that adjusts the entire hierarchy

↑ / ↓: Move to end of line above/below

← / →: Skip indentation areas
